django-faleconosco by gilsondev

Tenha o seu "Fale Conosco" pronto para o seu projeto.

Esse aplicativo foi desenvolvido para remover a tarefa de sempre criar uma página de contato para o seu site, portal ou blog. Com essa aplicação já tem tudo pronto como um formulário que tem: Nome, E-mail, Assunto e Mensagem. Ou seja, o básico que todo projeto para a web precisa.
